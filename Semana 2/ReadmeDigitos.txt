Para este modelo el dataset utilizad es basado en el porcentaje de precisión
1- Dataset utilizado para este ejercicio es load_digits de sklearn
2- inicialmente para las pruebas se asigno 20% para pruebas y validación y el 80% para entrenamiento.
3- Durante las ejecuciones se evidenció que el resultado del entrenamiento fue de 96.1 Vs el del modelo base que es 96.4 de precisión.
4- Se ajusto la asignación para pruebas de un 10% para ver el comportamiento.
5- Se evidencia que el resultado del entrenamiento es de 95.5 VS el modelo base de 94.4.
6- Aun que el modelo con el ajuste del 20% para pruebas y validaciones sea menor es un modelo que tiene bastante precisión.
